Todo lo que tu negocio necesita

Kitz entiende tu tipo de negocio — panadería, salón, tienda, restaurante, taller — y adapta su vocabulario y herramientas.

📦

Ventas y pedidos

“Vendí 5 arepas a $2 cada una” — Kitz captura la venta, calcula el total y lo registra automáticamente.

💰

Control de gastos

“Gasté $50 en harina” — registra gastos, consulta totales y genera resúmenes semanales.

📋

Inventario

Lleva el control de tus productos. Kitz te avisa cuando algo está por agotarse.

👥

Clientes

Registra clientes con nombre y teléfono. Programa seguimientos y mantén la relación.

📈

Insights con IA

Reportes semanales, resúmenes diarios y recomendaciones accionables generadas por IA.

🖨

Facturas

Genera facturas desde tus pedidos registrados. Envía por email directamente.

De cero a listo en 2 minutos

El onboarding es una conversación. Kitz te pregunta, tú respondes, y ya estás dentro.

Escríbele a Kitz

Manda un “Hola” al número de WhatsApp. Kitz se presenta y te pregunta qué tipo de negocio tienes.

Configura tu negocio

Dile si tienes panadería, salón, tienda, restaurante, taller, o lo que sea. Dale un nombre.

Registra tu primera venta

“Vendí 10 empanadas a $1.50” — Kitz entiende, registra y confirma. Ya estás operando.

Usa tu asistente a diario

Ventas, gastos, inventario, clientes, reportes — todo por chat, cuando quieras.

Empieza a desarrollar

Clona, configura variables de entorno, y levanta el gateway.

# Clonar el repositorio
git clone https://github.com/KenRoach/kitz.git
cd kitz/kitz_gateway_ts

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env

# Levantar en desarrollo
npm run dev

# O con Docker
docker build -t kitz .
docker run -p 8787:8787 kitz


API

# Health check
curl -s http://127.0.0.1:8787/v0.1/health

# Listar herramientas disponibles
curl -s http://127.0.0.1:8787/v0.1/tools

# Invocar una herramienta
curl -s -X POST \
  http://127.0.0.1:8787/v0.1/tools/sum/invoke \
  -H 'content-type: application/json' \
  -d '{"args":{"numbers":[1,2,3.5]}}'
